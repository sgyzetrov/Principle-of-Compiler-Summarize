## Keynote

编译过程概述（以翻译英文为例）

| 英文翻译过程               | 编译过程               |
| :------------------------: | :--------------------: |
| 识别出句子中的单词         | 词法分析               |
| 分析句子的语法结构         | 语法分析               |
| 根据句子的含义进行初步分析 | 语义分析及中间代码生成 |
| 对译文进行修饰             | 优化                   |
| 写出最后的译文             | 目标代码生成           |
